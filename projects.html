GovPulse
Completed Web

GovPulse is a FastAPI-based tool designed to monitor the status of government websites under .go.kr with low system load. It was created in response to the lack of a lightweight real-time recovery monitoring method during the government website outages following the National Information Resources Service fire on September 26, 2025

FastAPI

Python

Web Crawling

The project minimized server load through a low-overhead monitoring design using HEAD-first requests, partial GET fallback, and timeouts, improved practical usability by distinguishing informational notices from actual failures with a three-level status system

WildlifeVision Baseline — scikit-learn
Archived ML

Camera-trap image classification is highly sensitive to changes in shooting environments, and random train-test splits can easily overestimate real-world performance. To address this, the project extracts Histogram of Oriented Gradients (HOG) features from images, trains scikit-learn probabilistic classifiers, and uses GroupKFold validation to better reflect site generalization.

Python

scikit-learn

The model achieved a mean GroupKFold(site) log loss of 1.8639, recorded a leaderboard submission score of 1.9354, and was developed through 40 baseline and improvement experiments.
